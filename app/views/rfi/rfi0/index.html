CMA RFI Prototype

The links below illustrate ideas that will be further developed in a working solution for the CMA RFI project.

Authentication

For the administration interface, the system will use the GOV UK "One Login" system to authenticate users. For RFI responses, external users will use one-time access codes (for closed requests).
  • Login

Templates

RFI requests will be based on templates that can be re-used. A template sets out the content of the start page, the questions, validation rules, routing, and receipt email.
  • RFI Templates
  • Create new template
  • Edit existing template

Requests

When requests are sent (based on templates), they can be viewed. Responses will file themselves into dataverse automatically.
  • RFI Requests
  • RFI Responses
